# Customer-Shopping-Behaviour-Analysis

## Project Overview
This project analyzes customer shopping behavior to identify purchasing patterns, sales trends, customer preferences, and business insights using Python, SQL, and Power BI.

## Tools & Technologies
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- SQL
- Power BI
- Excel

## Project Workflow
1. Data Cleaning & Preprocessing using Python
2. Exploratory Data Analysis (EDA)
3. SQL-based Business Analysis
4. Interactive Dashboard Development in Power BI

## Key Analysis Performed
- Category-wise Sales Analysis
- Customer Demographics Analysis
- Seasonal Trend Analysis
- Subscription & Discount Impact
- Payment Method Analysis
- Shipping Preference Analysis

## Key Insights
- Clothing category generated the highest revenue.
- Customers aged 26–35 contributed more purchases.
- Winter season showed increased sales trends.
- Subscription users had higher spending patterns.
- Discounts positively impacted purchase frequency.

## Business Impact
The dashboard supports data-driven decision-making for:
- Marketing Strategy
- Inventory Optimization
- Customer Retention
- Seasonal Planning
- Sales Performance Monitoring
